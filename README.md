# Greenden – Responsive Website using HTML & Tailwind CSS

## Overview
Greenden is a fully responsive multi-page website built using HTML and Tailwind CSS.  
This project demonstrates how to create modern, clean, and flexible layouts using Tailwind’s utility-first approach.

It focuses on real-world website structure, responsive design principles, and professional UI layout practices.

## Technologies Used
- HTML5
- Tailwind CSS
- Responsive Web Design (Mobile-First Approach)

---

## Website Structure

### Home Page
- Clean and welcoming hero section
- Modern layout with proper spacing and typography
- Introduction to the Greenden concept

### Products Page
- Plant-themed product showcase
- Card-based grid layout
- Responsive alignment using Tailwind Grid and Flexbox

### Contact Page
- Structured contact information section
- Clean and organized layout for user interaction
- Consistent styling and spacing

---

## Responsive Design
- Fully responsive across mobile, tablet, and desktop devices
- Built using Tailwind CSS responsive breakpoints:
  - sm: Small devices
  - md: Medium devices
  - lg: Large devices
- Maintains consistent spacing, alignment, and typography across screen sizes
